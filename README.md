# TruX-Messenger
TruX — это инновационный текстовый мессенджер, воплощающий концепцию интеллектуального минимализма. Мы объединили предельно простой, интуитивно понятный интерфейс с мощными технологиями будущего. Главная особенность TruX — встроенная нейросеть AstraAI, которая превращает обычный чат в персональный центр управления информацией.
